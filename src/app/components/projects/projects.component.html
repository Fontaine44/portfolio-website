Used the shots data from MoneyPuck.com to generate shot heatmaps using Kernel Density Estimation. The data is updated automatically every night and uploaded to a database. Built a Python Flask API and an Angular front-end to visualize the shots.

Enhancing NHL Salary Evaluation through Dimensionality Reduction

This project explores the use of dimensionality reduction techniques to evaluate and analyze the salaries of NHL players. By applying methods such as PCA, PLS, Random Projection, and feature selection, the project aims to simplify the data while preserving important information. These techniques are assessed based on their ability to preserve model performance, reduce training time, and identify the most significant features in the dataset.
